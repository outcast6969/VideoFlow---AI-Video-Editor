# VideoFlow - AI Video Editor

Веб-приложение для обработки видео с использованием искусственного интеллекта. Позволяет применять различные стили обработки к видео для создания контента для социальных сетей.

Описание

VideoFlow анализирует загруженное видео и применяет к нему выбранный стиль обработки или выполняет кастомные инструкции пользователя. Приложение использует Claude API для анализа запросов и генерации параметров обработки, после чего применяет цветокоррекцию и эффекты к каждому кадру видео.

Технологии

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (Vanilla)
- Canvas API для обработки видео
- MediaRecorder API для создания видео
- Claude API (Anthropic) для AI-анализа
- File API и Drag & Drop API

Возможности

- Загрузка видео через интерфейс или перетаскивание файла
- Четыре предустановленных стиля обработки
- Кастомные текстовые инструкции для обработки
- AI-анализ запроса и создание плана обработки
- Покадровая обработка видео с применением фильтров
- Отображение прогресса обработки
- Скачивание обработанного видео

Установка

1. Склонируйте репозиторий:
bash
git clone https://github.com/outcast6969/videoflow.git


2. Перейдите в директорию проекта:
bash
cd videoflow


Запуск

Откройте файл index.html в браузере:

- Google Chrome (рекомендуется)
- Mozilla Firefox
- Safari
- Microsoft Edge

Альтернативный способ через локальный сервер:

Затем откройте http://localhost:8000 в браузере.

Использование

1. Откройте приложение в браузере
2. Загрузите видео файл (MP4, MOV, AVI, WebM)
3. Выберите один из предустановленных стилей или введите кастомные инструкции
4. Нажмите кнопку "Обработать видео с AI"
5. Дождитесь завершения обработки
6. Скачайте готовое видео

Стили обработки

- Вирусный контент - высокая насыщенность и контраст
- Эстетичный - мягкая пастельная цветокоррекция
- Кинематографичный - темные тона и высокий контраст
- Яркий и четкий - максимальная яркость и резкость

Технические детали

Приложение обрабатывает видео покадрово, применяя следующие параметры:
- Яркость (brightness)
- Контраст (contrast)
- Насыщенность (saturation)

Параметры генерируются AI на основе выбранного стиля или текстового запроса пользователя.

Ограничения

- Обработка ограничена первыми 30 секундами видео для оптимизации производительности
- Выходной формат - WebM
- Требуется современный браузер с поддержкой Canvas API и MediaRecorder
